# MC2-Team18-SungHozaza

<h1 align="middle">😴SungHozaza😴</h1>
<p align="middle">“수면 습관을 잡기 힘든 공시생들에게!” </p>
<p align="middle"> 스크린타임을 활용해 스마트폰을 제한하여 수면 습관을 잡아주는 서비스입니다!</p>

<p align="middle">"To test takers who are having a hard time getting sleeping habits!"</p>
<p align="middle">It's a service that uses ScreenTimeAPI to limit your smartphone to catch your sleeping habits!</p>

<br><br>
# 서비스 소개 (Service Description)
<p align="left">자야 할 시간에 스마트폰을 자주 사용하는 공시생이</p>
<p align="left">자야 할 시간에 자서 다음날 계획을 달성하도록 도와주는</p>
<p align="left">스크린 타임을 활용한 스마트폰 사용 제한 어플리케이션 입니다.</p>

<p align="left">Our service is a smartphone usage restriction application using screen time to help test takers who frequently use their smartphones when they need to sleep and achieve their plans for the next day</p>
